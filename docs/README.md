# GraphMind v2

MCP-сервер памяти для AI-агентов. Хранит факты, причинно-следственные связи и правила в графе знаний, доступном через Model Context Protocol.

## Возможности

- **Иерархическая память** — L0 (кластеры) → L1 (домены) → L2 (атомарные факты)
- **Причинно-следственный граф** — связи «причина → следствие», поиск корней и последствий
- **Многопользовательские workspace'ы** — изоляция памяти между проектами
- **Векторный поиск** — семантический поиск по содержимому памяти
- **Планировщик задач** — P0 → P1 → P2 → P3 иерархия планов с декомпозицией
- **Trust Firewall** — верификация входящих данных по источнику и консистентности

## Быстрый старт

```bash
# Docker
docker compose up -d

# Локально
cargo run --release
```

MCP-сервер доступен через stdio. Подключите его в вашем MCP-клиенте (Kodik, Claude Desktop, и др.).
